# 📊 GPA Calculator

A simple, responsive web application designed to help college students quickly calculate their semester Grade Point Average (GPA)[cite: 1]. Built using standard front-end technologies, this project computes real-time results directly within the web browser[cite: 1].

## ✨ Features
* **Dynamic Course Rows:** Add as many courses as you need with a single click[cite: 1].
* **Instant Calculation:** Computes your GPA instantly using weighted credit hours[cite: 1].
* **Responsive Design:** Clean, modern interface that works perfectly on both desktop and mobile screens[cite: 1].
* **Serverless Execution:** Runs completely on the client side with no backend database setup required[cite: 1].

## 🛠️ Tech Stack
* **HTML5:** Structures the layout, input fields, and UI elements[cite: 1].
* **CSS3:** Handles clean, modern styling and flexible layouts[cite: 1].
* **JavaScript (ES6):** Controls the core mathematical logic and dynamic DOM manipulation[cite: 1].

## 🚀 How to Run Locally
1. **Clone or Download** this repository to your local machine[cite: 1].
2. Navigate to the project directory[cite: 1].
3. Double-click the `index.html` file to open it instantly in any modern web browser (Chrome, Firefox, Safari, Edge)[cite: 1].

## 🧮 Calculation Formula
The application calculates your final score using the weighted average formula[cite: 1]:

$$\text{GPA} = \frac{\sum (\text{Grade Points} \times \text{Credits})}{\sum \text{Total Credits}}$$
